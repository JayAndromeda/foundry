---
title: Tooltip
description: A popup that displays non-essential information related to an element when the element receives keyboard focus or the mouse hovers over it.
---

<ComponentPreview>
  <Tooltip content="Add to wishlist">
    <Button>+</Button>
  </Tooltip>
</ComponentPreview>

## Usage

```tsx
import { Tooltip } from "@foundry-ui/Tooltip";
```

```tsx
<Tooltip content="Add to wishlist">
  <Button>+</Button>
</Tooltip>
```
