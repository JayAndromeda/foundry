## Usage

```scss
.element {
  background: var(--grayBg);
  color: var(--grayText);
  transition: 150ms ease;
  transition-property: background, color;

  &:hover {
    background: var(--grayBgHover);
    color: var(--grayTextContrast);
  }
}
```
